Description

atom_keys = AtomPrompt4 [ 'prompt_token', "mode_string" ]

Prompts for atoms in the MOE Window and returns the keys of the selected atoms that fit the specified mode_string.

'prompt_token'
Prompt message that immediately precedes Pick first atom... at the CLI.
"mode_string"
Describes the required bonding status between successive atoms, and is a string with any combination of the following characters:
  • "n": nonbonded
  • "b": bonded
  • "r": bonded by rotatable bond

The user will be prompted to select a number of atoms which is one more than the length of mode_string. AtomPrompt4 will not accept a selection that does not respect the specified bonding mode. Should this be the case, MOE will emit a beep.

The following example prompts the user to select only atoms joined by a dihedral angle:

     AtomPrompt4 [ 'Select 4 dihedral atoms', "brb" ]

DihedralEnergyPlot [ [a_key, b_key, c_key, d_key], num_samples ]

Plots the potential energy of the molecular system against rotational increments of a specified dihedral angle. Values are displayed in a plot panel.

dihedral_atoms
Vector of 4 atom keys. If no atom keys are given ( [] ), the user is prompted to select atoms in the MOE Window.
num_samples
Integer that specifies the number of equally spaced increments through which the dihedral angle will be rotated. The default value of num_samples is 360 degrees/5, and the minimum accepted value is 4.
DihedralEnergyContour [ [dihedral_atoms, dihedral_atoms], num_samples ]

Creates a contour plot of the potential energy of the molecular system against rotational increments of two specified dihedral angles. Values are displayed in a plot panel.

dihedral_atoms
Two vectors each containing 4 atom keys. If no atoms are given (if dihedral_atoms is [] for one or both of the specified angles), the user is prompted to select atoms in the MOE
num_samples
Integer that specifies the number of equally spaced increments through which both the dihedral angles will be rotated. The default value of num_samples is 360 degrees/10, and the minimum accepted value is 4.
MeasurePotential []

Calculates the value of the potential function of the molecular system and displays the results in the MOE window. The total energy is displayed as well as the contributions to the energy of the bond length, bond angle, torsions, electrostatic and van der Waals terms. Furthermore, if any atoms are selected, the energies between the selected and non selected atoms are displayed.

MeasureDistances []

Prompts for pairs of atoms and creates distance meters between them.

MeasureAngles []

Prompts for triplets of atoms and creates angle meters between them.

MeasureDihedrals []

Prompts for quartets of atoms and creates dihedral meters between them.

Note: The three previous commands prompt for atoms and are used to calculate, create and display meters in the MOE Window. Each consists of an infinite loop and is thus of no practical use within SVL programs. They are called by commands in the Measure page of the right button bar in the MOE window.

RemoveDistanceMeters []

Removes all distance meters from the system.

RemoveAngleMeters []

Removes all angle meters from the system.

RemoveDihedralMeters []

Removes all dihedral meters from the system.
