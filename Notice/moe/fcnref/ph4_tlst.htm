Pharmacophore Typing Functions

Syntax

    flags    = ph4_aAcceptor         atom_keys
    flags    = ph4_aAcid             atom_keys
    flags    = ph4_aAnion            atom_keys
    flags    = ph4_aBase             atom_keys
    flags    = ph4_aCation           atom_keys
    flags    = ph4_aDonor            atom_keys
    flags    = ph4_aGeometry         atom_keys
    flags    = ph4_aGrease           atom_keys
    flags    = ph4_aHydrophobe       atom_keys
    flags    = ph4_aLinear           atom_keys
    flags    = ph4_aPlanar           atom_keys
    flags    = ph4_aResonanceAnion   atom_keys
    flags    = ph4_aResonanceCation  atom_keys
    flags    = ph4_aTautomerAcceptor atom_keys
    flags    = ph4_aTautomerDonor    atom_keys

    alltypes = ph4_TypeList          []
    types    = ph4_aType             atom_keys

Description

The ph4_a* functions return, for each atom in the argument, 1 if the atom is of the specified pharmacophoric type, 0 otherwise.

Function Description
ph4_aAcceptor Returns 1 for a proton acceptor. Does not take protonation state or tautomerism into account.
ph4_aAcid Returns 1 for an acidic atom (i.e. that would be deprotonated when washed).
ph4_aAnion Returns 1 for an atom that has localized excess negative charge (an acid or an anion).
ph4_aBase Returns 1 for a basic atom (i.e. that would be protonated when washed).
ph4_aCation Returns 1 for a given atom with localized excess positive charge (a base or a cation).
ph4_aDonor Returns 1 for a proton donor. Does not take protonation state or tautomerism into account.
ph4_aGeometry Returns 0 for isolated atoms, 1 for linear (sp) atoms, 2 for linear (sp2) atoms, and 3 for other atoms.
ph4_aGrease Returns 1 for an atom with very hydrophobic character.
ph4_aHydrophobe Returns 1 for a hydrophobic atom.
ph4_aLinear Returns 1 for linear (sp) atoms.
ph4_aPlanar Returns 1 for planar (sp2) atoms.
ph4_aResonanceAnion Returns 1 for an atom that exists in a resonance structure in which it is an anion.
ph4_aResonanceCation Returns 1 for an atom that exists in a resonance structure for which it is a cation.
ph4_aTautomerAcceptor Returns 1 for an atom that can be a proton acceptor due to tautomerism.
ph4_aTautomerDonor Returns 1 for an atom that can be a proton donor due to tautomerism. 
ph4_TypeList []

Returns all the legal pharmacophore types that were available before the advent of the ph4_a* functions. These pharmacophore types comprise a simpler set of labels than those expressible with the ph4_a* functions. The pharmacophore types are returned as tokens:

'-' anion (acidic functionality)
'+' cation (basic functionality)
'A' H-bond acceptor
'D' H-bond donor
'H' hydrophobe
'P' polar (both H-bond donor and acceptor, e.g. H2O)
'X' other

Note: ph4_aType can be used to obtain these values for specified atoms (see below).

ph4_aType atom_keys

Returns the atom pharmacophore type of each atom specified in atom_keys. The possible types are as returned by ph4_TypeList.

References

Sheridan, R.P.; Bush B.L.; PATTY: A programmable atom typer and language for automatic classification of atoms in molecular databases; J. Chem. Info. Comput. Sci. 33 (1993) 756–62.
