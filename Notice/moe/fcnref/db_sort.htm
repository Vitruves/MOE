MOE Database Sorting Functions: db_Sort, db_Proxy

Syntax

    mask = db_Sort [ mdb, fields, sort_order_flag ]

    proxies = db_Proxy [ mdb, fields, sort_order_flag ]

Description

mask = db_Sort [ mdb, fields, sort_order_flag ]

Sorts the molecular database according to the values of the specified fields.

mdb
Key or filename of the molecular database on which the sort operation will be performed.
fields
Names of the fields to sort on, expressed as a vector of tokens.

If more than one field is specified, the first field is used as the primary sorting criterion. Subsequent fields are used as tie breakers in the event of identical data values in the primary sort field.

sort_order_flag
Specifies the sorting direction for each specified field: 0 = ascending, 1 = descending.

In the following example, entries in the database are first sorted according to the descending order of values in 'field1'. The second field, 'field2', is used to sort identical values in 'field1' by ascending order.

    db_Sort [ mdb, ['field1', 'field2'], [1,0] ]

For each entry in the database, the function returns a 1 if it is a unique value i.e., the first instance of the value if it appears in more than one cell, and 0 otherwise. For instance, if the field being sorted in ascending order contains [2,3,4,2], db_Sort will sort the field so that it is [2,2,3,4] and return [1,0,1,1].

proxies = db_Proxy [ mdb, fields, sort_order_flag ]

Returns a vector of numerical scalars which represent the sort order (or rank) of the database entries based on the specified fields. These scalars can then be used as substitute entry values by which to sort.

mdb_key
Key or filename of the molecular database.
fields
Names of the fields to sort on, expressed as a vector of tokens.

If more than one field is specified, the first field is used as the primary sorting criterion. Subsequent fields are used as tie breakers in the event of identical data values in the primary sort field.

sort_order_flag
Specifies the sorting direction for each specified field: 0 = ascending, 1 = descending.
