Rotation Functions: rot3d_Print

Syntax

    rot3d_Print M

Description

Echoes the matrix M to the SVL Commands window output display area. M is expected to be given as a vector of row-vectors; i.e. the matrix:

a1 a2 a3
b1 b2 b3
c1 c2 c3

is supplied as:

    [ [a1,a2,a3], [b1,b2,b3], [c1,c2,c3] ]

Returns nothing.
