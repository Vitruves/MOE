License Information Function: LicenseInfo

Syntax

    LicenseInfo []
    data = LicenseInfo []

Description

Returns information about the license file. The license file limits the usage of the licensed application, specifying the highest version of the software that can be run within the duration of the license, and the maximum number of copies that can be run simultaneously.

The returned data is a tagged vector with the following information:

path Path to the license file in use.
feature Name of the licensed application.
version Highest numbered version allowed.
expires Expiry date, given as dd-mmm-yyyy. If year is 0000, the feature never expires.
key Hexadecimal authentication key used to verify that the license has not been modified.
short_key First 16 characters of full key.
server Hostname or IP address of the computer acting as license server.
ntokens Total number of tokens hosted by the license server. A value of 0 means unlimited.
nused Total number of tokens currently in use. A value of 0 means unlimited.
nfloatused Number of floating tokens currently in use. Floating tokens are not limited to a specific machine or user and can be used by multiple different users in turns. A value of 0 means unlimited.
days Days until the license file expires.
notice Notice text from the license file.
user Current user and host information.
hostid Host identification information.
hostid_ether Network port host identification information.
display Host display information.
flexnet Flexnet MOE version information.
daemon Flexnet license server version information.
timeout License server timeout.

For example:

    svl> x = LicenseInfo []
    svl> x
    [ path: 'C:/moe/license.dat', feature: 'moe', version: '2020.03',
      expires: '31-may-2020', server: 'lic.chemcomp.com', ntokens: 6, ... ]

When the return value of LicenseInfo is not assigned to a variable, the function prints a report about the license.

    svl> LicenseInfo []
     Path    : C:/moe/license.dat
     Feature : moe
     Version : 2020.03
     Expires : 31-mar-2020 (11 days left)
     Key     : 1234 ABCD 1234 AB4D 1FEG 1234 ABCD 1234 AB4D ...
     Server  : lic.chemcomp.com
     Tokens  : 6 (3 used)
     Notice  : 
     User    : USER=moeuser HOSTNAME=computer1
     HostID  : 12ab34cd56e1
     Display : DISPLAY=
     FlexNet : v11.14.1.2 build 208719 x64_lsb
     Daemon  :
