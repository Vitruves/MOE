Clipboard Functions

Unsupported Functions

Syntax

    ClipboardSetData tag_v
    char_data = ClipboardGetData fmt_tok

    ClipboardSetText str
    str = ClipboardGetText []

    fmt_toks = ClipboardTags []

Description

ClipboardSetData tag_v

Puts data of different formats onto the clipboard. Any data previously stored on the clipboard is first cleared. The data is supplied as a tagged vector, where the tags are format tokens specifying the formats of the data items, and the attributes are character (string) data.

Here is an example:

    ClipboardSetData [ smiles: "c1ccccc1C=O", text:"Hello world!" ]

char_data = ClipboardGetData fmt_tok

Returns the data of the specified format from the clipboard. fmt_tok is a scalar value and the data is returned as characters. By default, 'text' is always a valid format even if no text has actually been placed on the clipboard.

ClipboardSetText str

Puts the text string str onto the clipboard. The implied data format is 'text'.

str = ClipboardGetText []

Gets text data from the clipboard; the text data is returned as a string. The implied format is 'text'.

fmt_toks = ClipboardTags []

Returns the formats (as tokens) of the data currently stored on the clipboard. By default, 'text' is always a valid format, even if no text data has been placed on the clipboard.
