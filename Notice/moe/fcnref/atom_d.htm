Atom Display Functions

Syntax

    bond_tokens    = aBondLook      atom_keys
    colors         = aColor         atom_keys
    colorby_tokens = aColorBy       atom_keys
    radii          = aDisplayRadius atom_keys
    flags          = aHidden        atom_keys
    label_tokens   = aLabel         atom_keys
    flags          = aLabelHidden   atom_keys
    nuc_tokens     = aNucleusLook   atom_keys
    colors         = aRGB           atom_keys
    numbers        = aScalar        atom_keys
    flags          = aSelected      atom_keys

    aSetBondLook     [atom_keys, bond_tokens]
    aSetColorBy      [atom_keys, colorby_tokens]
    aSetHidden       [atom_keys, flags]
    aSetLabel        [atom_keys, label_tokens]
    aSetLabelHidden  [atom_keys, flags]
    aSetNucleusLook  [atom_keys, nuc_tokens]
    aSetRGB          [atom_keys, colors]
    aSetScalar       [atom_keys, numbers]
    aSetSelected     [atom_keys, flags]

Description

In MOE, molecular data is organized as a hierarchy of molecular objects or objects. MOE stores atom information in separate internal data structures. Atoms are referenced with an object key, which is a unique number identifying the atom object. The many atom functions provided in MOE cover the following broad categories:

  • Fundamental Properties
  • Potential Properties
  • Display Properties (this page)
  • Geometry Properties

The following functions can be used to obtain information from atom objects:

Function Description
aBondLook
 For each atom in the argument, this function returns a token describing how the atom's bonds are rendered. Legal values are defined in the constant A_BOND_LOOKS and consist of 'none', 'line' and 'cylinder'. This function can be used on arbitrarily nested vectors.
aColor Returns the display color for the atom. This is the (computed) color that is actually used for the atom. The integer color format is 0xRRGGBB encoding red, green and blue.
aColorBy For each atom in the argument, this function returns a token describing how the atom is colored. Legal values are
  • 'element' - color by atom's element
  • 'scalar' - color by atom's aScalar value
  • 'rgb' - color by atom's aRGB color
  • 'tempfactor' - color by the atom's temperature factor
  • 'a:rgb' - color by atom's aRGB color
  • 'r:rgb' - color by the owning residue's rRGB color
  • 'c:rgb' - color by the owning chain's cRGB color
  • 'r:aseg' - color by the owning residue's secondary structure color
  • 'c:number' - color by the owning chain's index color
  • 'chain' - color by the owning chain's display color
  • 'residue' - color by the owning residue's display color
  • 'ribbon' - color by the owning residue's ribbon display color
This function can be used on arbitrarily nested vectors.
aDisplayRadius
 Returns the atom's radius (in angstroms) used for rendering. Points have a radius of 0.
aHidden
 For each atom in the argument, this function returns a value of 1 if the atom is hidden in the MOE Window, and zero otherwise. This function can be used on arbitrarily nested vectors.
aLabel For each atom in the argument, this function returns the atom label token. This function can be used on arbitrarily nested vectors.

Atom label text supports HTML tags for text formatting and positioning. Atom property values such as atom name, element, charge, etc. can be added via shortcuts.

aLabelHidden
 For each atom in the argument, this function returns a value of 1 if the atom label is hidden in the MOE Window, and zero otherwise. This function can be used on arbitrarily nested vectors.
aNucleusLook
 For each atom in the argument, this function returns a token describing how the atom's nucleus is rendered. Legal values are defined in the constant A_NUCLEUS_LOOKS and consist of 'none', 'point', 'sphere' and 'small-sphere'. This function can be used on arbitrarily nested vectors.
aRGB Returns the internal color of the atom. This is the color used when the atom's ColorBy mode is rgb or a:rgb. The internal color is an integer representing the RGB color in the format 0xRRGGBB encoding red, green and blue.
aScalar Returns the "atom scalar" number of each atom in the argument. The scalar value, a floating point value between 0 and 1, is used to color atoms when the ColorBy mode is 'scalar'. This function can be used on arbitrarily nested vectors.
aSelected For each atom in the argument, this function returns a value of 1 if the atom is selected in the MOE Window, and zero otherwise. This function can be used on arbitrarily nested vectors.

The following functions can be used to modify information in atom objects:

Function Description
aSetBondLook Sets the 3D display mode of bonds for each atom in the argument. Legal values are defined in the constant A_BOND_LOOKS and consist of 'none', 'line' and 'cylinder'. This function can be used on arbitrarily nested vectors.
aSetColorBy Sets the coloring mode (for 3D display) for each given atom. The scalar value is used to color atoms when the ColorBy mode is 'scalar'. Legal values are
  • 'element' - color by atom's element
  • 'scalar' - color by atom's aScalar value
  • 'rgb' - color by atom's aRGB color
  • 'tempfactor' - color by the atom's temperature factor
  • 'a:rgb' - color by atom's aRGB color
  • 'r:rgb' - color by the owning residue's rRGB color
  • 'c:rgb' - color by the owning chain's cRGB color
  • 'r:aseg' - color by the owning residue's secondary structure color
  • 'c:number' - color by the owning chain's index color
  • 'chain' - color by the owning chain's display color
  • 'residue' - color by the owning residue's display color
  • 'ribbon' - color by the owning residue's ribbon display color
This function can be used on arbitrarily nested vectors.
aSetHidden Sets the hidden flag for each given atom. Atoms with a true hidden flag will not be displayed in the MOE Window. This function can be used on arbitrarily nested vectors.
aSetLabel Sets the label for each given atom. Atom labels are displayed in the MOE Window. This function can be used on arbitrarily nested vectors.

Atom label text supports HTML tags for text formatting and positioning. Atom property values such as atom name, element, charge, etc. can be added via shortcuts.

aSetLabelHidden Sets the hidden flag of the atom label for each given atom. Setting the hidden flag to true will cause the label of the associated atom to no longer be shown in the MOE Window. This provides the ability to hide the atom label without deleting it. This function can be used on arbitrarily nested vectors.
aSetNucleusLook Sets the 3D display mode of the nucleus for each given atom. Legal values are defined in the constant A_NUCLEUS_LOOKS and consist of 'none', 'point', 'sphere' and 'small-sphere'. This function can be used on arbitrarily nested vectors.
aSetRGB Sets the internal color of the atom. This is the color used when the atom's ColorBy mode is rgb or a:rgb. The internal color is an integer representing the RGB color in the format 0xRRGGBB encoding red, green and blue.
aSetScalar Sets the scalar (floating point) value for each given atom. The numbers must be between [0,1]. Values outside this range will be clamped. This function can be used on arbitrarily nested vectors.
aSetSelected Sets the selected flag for each given atom. Selected atoms are marked in the MOE Window. This function can be used on arbitrarily nested vectors.

Obsolete Syntax

flags = aLabelChain   atom_keys
flags = aLabelCharge  atom_keys
flags = aLabelElement atom_keys
flags = aLabelMMType  atom_keys
flags = aLabelName    atom_keys
flags = aLabelRes     atom_keys

For each atom in the argument, this function returns a value of 1 if the atom label component is enabled, and zero otherwise. This function can be used on arbitrarily nested vectors.

Superseded by function aLabel.
