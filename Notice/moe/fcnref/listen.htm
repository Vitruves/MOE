Port Listening Functions

Syntax

    oldport = listen_port port

Description

This function is used to start a listener socket on the specified TCP port and wait for incoming remote HTTP connections. HTTP requests can be made to call predefined functions remotely, described in more detail in the section on the MOE Socket Listener.

WARNING: This feature allows any user on the network to send commands to another user's machine, with no authentication required. For this reason, use this feature only on a trusted network!

oldport = listen_port port

This function attempts to open a listening socket on the specified TCP port. Valid values of port are positive integers, zero, or null.

If port is a positive integer, the listener will begin to listen to the specified port. If the listener was already listening to another port, it will switch from that port to the specified one. If an error occurs trying to listen to the specified port, the previous state of the listener is preserved; i.e. the listener continues to listen to the previous port. If port is zero, listening is shut down completely. If port is null, the function does nothing. In all cases, the value of the port previously being listened to (prior to calling the function) is returned.
