Molecule Input Language Functions: sm_StringRoot

Syntax

    elements = sm_StringRoot [ SMILES_token ]
    elements = sm_StringRoot [ SMILES_token, dkeys ]

Description

For each molecular fragment represented by a molecular input language string in the token vector SMILES_token, this function returns the element of the first atom. If no element is specified, or if a wildcard is encountered, '*' is returned. $() expressions (atom environments) are permitted, as is the use of symbolic links. It is an error to use a symbolic link (see sm_Match, sm_Define, sm_Create) without passing the key of the dictionary in which the referenced entry is to be found.

    svl> sm_StringRoot 'O=C1CCC(=O)CC1'
    'O'
    svl> sm_StringRoot [ ['[#6]','n1aaaa1','A'] ]
    ['C','N','*']

For details on syntax, see sm_Build and sm_Match.
