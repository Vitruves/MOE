Description

The constants ELEMENT_COL, ELEMENT_ROW, ELEMENT_TMETAL and ELEMENT_SYM are each indexed by an atomic number and contain, respectively, the periodic table column number, the periodic table row number, a flag which indicates whether the element is a transition metal, and the element symbol token.

The constant EL_GEOMETRIES defines legal orbital geometries: 'sp', 'sp2', 'sp3', 'dsp3', 'd2sp3', 'd3sp3'.

These functions take a vector of element tokens and return properties of the elements as defined in the element database.

Function Description
el_COV_Radius Returns the covalent radius (in Å). 
    svl> el_COV_Radius ['He','C']
    [0.37,0.68]
el_VDW_Radius Returns the van der Waals radius (in Å). 
    svl> el_VDW_Radius ['He','C']
    [1.22,1.85]
el_DefaultMass Returns the default mass in atomic mass units (AMU). 
    svl> el_DefaultMass ['He','C']
    [4.0026,12.011]
el_Protons Returns the default number of protons. 
    svl> el_Protons ['C','Au','CH1']
    [6,79,7]
el_Elements Returns the symbol of each element in the periodic table (including the lone pair pseudo-element) expressed as tokens.
el_Metal
el_NonMetal
el_SemiMetal		 Return 1 if the corresponding element is a metal, non-metal, or semi-metal, respectively, 0 otherwise.
el_Valence Returns the default number of valence electrons. 
    svl> el_Valence ['H','N','Br']
    [1,5,17]
el_spValence Returns the default number of sp valence electrons. 
    svl> el_spValence ['H','N','Br']
    [1,5,7]
el_MaxValence Returns the default maximum number of valence electrons. 
    svl> el_MaxValence ['H','N','Br']
    [2,8,18]
