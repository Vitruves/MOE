Atom Functions: SelectedAtoms, SelectedTorsion,...

Syntax

    atom_keys = SelectedAtoms []
    atom_keys = SelectedTorsion []

    residue_keys = SelectedResidues []
    chain_keys = SelectedChains []

Description

atom_keys = SelectedAtoms []

Returns the keys of all the selected atoms in the system.

atom_keys = SelectedTorsion []

If there are exactly four selected atoms in the system which are related by torsion, SelectedTorsion[] returns the keys of the four atoms in order along the bond (1-2-3-4). Otherwise, returns [].

residue_keys = SelectedResidues []

Returns the keys of all the selected residues in the system.

chain_keys = SelectedChains []

Returns the keys of all the selected chains in the system.
