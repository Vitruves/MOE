Description

chain_keys = dna_Create[ residue_names, opt ]

options = [
    conformation : structure_token
]

Creates a double-stranded DNA/RNA molecule in a specified conformation and returns the keys of the two newly created chains. The nucleoside composition of the first chain will be as specified by dna_Create while the second chain will be the complement chain as dictated by Watson-Crick base-pairing rules. In the Sequence Editor, both the sequence of the main chain and the sequence of the Watson-Crick base-pairing complement chain are written in the 5' to 3' direction.

residue_names
Vector of nucleic-acid tokens specifying the nucleoside composition:
Code Nucleic Acid
A r-Adenosine
C r-Cytidine
G r-Guanosine
U r-Uracil
DA d-Adenosine
DC d-Cytidine
DG d-Guanosine
DT Thymidine
The options parameter is given as a list of attribute-value pairs:
conformation: One of the standard DNA conformations expressed as a token (case sensitive):
  • 'A'
  • 'B'
  • 'C'
  • 'Z'
  • 'A*'
  • 'B*'
  • 'C*'

The default is 'B'.

A, B, and C-Helix conformations are available in two different variants. The first variant with parameters provided by the w3DNA server [Zheng 2009] [LU 2003]. The other form uses the published parameters from Arnott [Arnott 1972] [Arnott 1975], these variants are labelled A*,B*,C*.

For example, the following function:

svl> dna_Create [ ['DG','DT','DA'], [conformation: 'A'] ]

returns the keys of both chains. The Sequence Editor will show the DNA sequence dG,dT,dA as well as the complement sequence dT,dA,dC.

dna_Builder []

Opens the DNA/RNA Builder panel, which allows building single and double-stranded DNA/RNA molecules.

Obsolete Functions

chain_keys = DNA_Create[ "nucleosides", 'conformation' ]

Creates a double-stranded DNA molecule in the specified conformation and returns the keys of the two newly created chains. The nucleoside composition of the first chain will be as specified by DNA_Create while the second chain will be the complement chain as dictated by Watson-Crick base-pairing rules. In the Sequence Editor, both the sequence of the main chain and the sequence of the Watson-Crick base-pairing complement chain are written in the 5' to 3' direction.

nucleosides
String of one-letter codes specifying the nucleoside composition:
1-Letter Code Nucleic Acid
A Adenosine
C Cytidine
G Guanosine
T Thymidine

One-letter codes are not case sensitive.

conformation
One of the standard DNA conformations [Arnott 1972] [Arnott 1975] expressed as a token (case sensitive):
  • 'A'
  • 'B'
  • 'C'

The default is 'B'.

For example, the following function:

svl> DNA_Create [ "GTA", 'A' ]

returns the keys of both chains.The Sequence Editor will show the DNA sequence dG,dT,dA as well as the complement sequence dT,dA,dC.

DNA_Builder []

Opens the old DNA Builder panel, which allows building double-stranded DNA molecules using the function DNA_Create.
