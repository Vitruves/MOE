Overview

The Atom Labels panel allows to create new or change existing atom labels. It is opened by choosing MOE | Render | Atom Labels. Alternatively, it can be opened from the Atom Render panel (MOE | Render | Atoms or MOE | Footer | Atoms) by pressing the button. The Atom Labels panel can be attached or detached at the left side of the MOE main window with the Attach/Detach button in the panel footer.

Manage Atom Label Selection

All functions in the Atom Labels panel operate on the currently selected atoms. The number of selected atoms is shown at the top of panel. If no atoms are selected this selection count will be shown in red, to remind the user that atoms need to be selected for atom label creation or changes.

Right below the selection count are action buttons to help focus on the current selected atoms/labels. One can Center the view on the current selected atoms, Isolate them by hiding all unselected atoms, or make sure to Show all selected atoms (Labels are only displayed on visible atoms).

Select Root Atoms replaces the current atom selection with one selected representative atom per residue. This is useful, if one wants to e.g. label residue or chain names on only one atom per residue. Pressing the button will choose and select one such representative atom (e.g. CA for amino acids or C1' for nucleic acids) in every residue with selected atoms.

All unique atom labels are shown in the Labels list. The number of atoms which share that label is shown in the Atoms column. The visibility of the label is displayed as well (S/H: shown/hidden). Selecting a label in the list will select all atoms with that label. Only one unique label can be selected in the list at a time. If atoms with different labels are selected in the MOE Window, the label list entry with the most selected atoms will be selected in the list.

Note that the label list shows the labels without formatting tags. Thus, two labels with same text but different formatting will appear in the list twice with the same name. The same applies to visibility, if the same label is shown on some atoms and hidden on others.

Show/Hide Atom Labels

 The labels of selected atoms can be hidden and shown without deleting the atom labels themselves using the Show and Hide buttons. Show All and Hide All will show/hide the labels of all atoms, respectively. Atom labels can also be hidden and shown from the Atom Render panel in the footer with the and buttons.

Create/Edit Atom Labels

If an existing unique label is selected in the label list, the label text is accessible and can be changed in the Edit Label text field. If atoms without an atom label are selected, the text field is empty and one can start typing a new atom label text. In both cases all changes to the atom label text are automatically applied onto all selected atoms. The text field allows to create multi-line labels by just adding carriage returns as needed.

If one has atoms selected with one unique label, but also atoms without label, the Edit Label text is colored blue, to indicate that there are atoms which have not yet this label set. Additionally the Apply button is now active, if one presses it, the current label text will be set on all selected atoms.

If atoms with different labels are selected, the majority label text is shown in the text field and colored red. If one presses Apply or changes the label all selected atoms will get the new label text assigned, potentially overwriting different existing atom labels.

Note! The label text can be further formatted with the buttons below the edit text field. The formatting is done through HTML tags in the label text, which also can be entered manually in the text field. Due to the use of HTML syntax the two characters "<" and "&" are considered special characters and can not be used directly in atom label text, to use either of them explicitly, one has to specify their HTML equivalents "&lt;" and "&amp;". They can be entered manually or with the Insert popup panel.

Insert Label Shortcuts

It is possible to add common atom or residue property values such as atom names, element symbols, residue names, etc. into the atom label text. This is done with the Insert button, which will open a subpanel to choose from the available atom, residue, and chain properties. The properties are represented in the label text field as a named shortcut enclosed in curly brackets (e.g. {aName} for atom names). The shortcut is then replaced by the current value of the atom, residue, or chain property in the actual 3D atom label:

Name:{aName} → Name:HB12

A complete list of atom label shortcuts is available below.

The subpanel list also includes several common text symbols, like the Ångström character (Å), which can also be inserted. The list of available symbols can be found here.

The shortcut/symbol list can be searched with the Filter text field. A selected shortcut or symbol is inserted into the label text at the current cursor position with either a double click on the list item or pressing the OK/Insert buttons. OK will also close the shortcut list subpanel.

Tip! The display width and precision of floating point property values, provided by shortcuts, can changed by adding width.precision specifiers preceded by a colon into the shortcut in the label text, e.g.:

Charge:{aCharge}      → aCharge:0.342
Charge:{aCharge:5.2f} → aCharge: 0.34

Similar syntax can be used to limit the output length of text properties like chain name or header:

Chain:{cName}     → Chain:1ABC.A
Chain:{cName:.3s} → Chain:1AB

Note! Since the curly brackets are used to delimit shortcut definitions one needs to use a double open bracket "{{" if one wants to add a literal open curly bracket to an atom label.

Format Label Text

The label text can formatted, showing parts of the text in e.g. bold, different colors, or different font styles. To format a part of the text, select it in the Edit Label text field and then press the requested font style button. If nothing is selected, the format of the whole label text is changed. The formatting change is immediately shown on the atom label in the 3D main window. The bubblehelp of the text field also displays the label text with formatting.

The formatting is stored in the label text in the form of HTML-like tags (e.g. <b>,<i>,<sub>,etc.). The formatting tags are by default hidden in the edit text field, but can be displayed by switching the Show HTML checkbox on. In this mode the HTML formatting can be added or changed in the label text directly.

While editing an atom label, all text and format changes are recorded and can be undone/redone with either the provided buttons, or pressing the Ctrl-Z or Ctrl-Y keys in the edit text field. One can also remove all formatting from a label text or delete the label text completely:

Set Atom Label Position

Atom labels are positioned/attached by default to the upper right of the labelled atom (or centered for atoms in CPK style). This default attachment position can be changed. It is also possible to freely position atom labels with the mouse. Additionally one can emphasize atom labels by putting them into a text bubble/balloon attached to the atoms.

Button Description HTML Tag
Fill Put the atom label into a text balloon attached to the atom and set the background color of the text balloon. Press x to remove the label balloon.

Tip! Balloon labels can be picked with the mouse. If such a label is picked, the associated atom will be selected.

<pos bg=00ff00> </pos>
Set a custom atom label attachment position from top-left to bottom-right. Press x to clear and use the default label attachment position. <pos l=1> </pos>

1: top-left, ... 9: bottom-right

Start and stop manual label positioning.

When enabled, one can drag the atom labels of the currently selected atoms with the left mouse button in the MOE main window. The manually set label position is still relative to the atom position. If the atom is moved (e.g. by a minimization calculation), the label will move the exact same distance and direction as the atom.

<pos x=1 y=2.3> </pos>
Set label offset in Z direction to optimize the position of atom labels in front or back of an atom. <pos z=0.5> </pos>
x Reset atom label position to default.

Configure Atom Quick Labels

The Atom Render panel (MOE | Render | Atoms) provides quick access to six standard atom labels: Element, MM, Charge, Name, Residue, and Chain. With the Manage button in the footer of the Atom Labels panel one can add, remove, and customize these Quick Label buttons. It opens the Configure Quick Labels panel which allows to define up to nine default or customized Quick Label buttons:

Button Title Set Quick Label button title. Choose with the down arrow from one of the six default Quick Label button titles (Element, MM, Charge, Name, Residue, Chain) or select to enter a custom button title/action (Custom).
Label String Enter the HTML formatted atom label string to be assigned to the Quick Label button. Click down arrow to copy the current label text definition from the Atom Labels panel.
Residue If enabled, then this label will be applied to only one representative atom per residue when either all residue atoms or no atoms at all are selected. This is useful for per-residue labels, e.g. residue name.
OK Save changes and close configuration panel.
Default Set all settings to MOE default values.
Close Discard changes and close config panel.
