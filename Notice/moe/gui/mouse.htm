Using a One, Two or Three-Button Mouse

In MOE, both the keyboard and mouse may be used to perform operations such as selecting atoms and choosing menu commands. However, the mouse is necessary to perform actions such as rotating the molecular system in the MOE rendering area.

The type of mouse used may vary with operating system or preference. For consistency the MOE documentation assumes a right-handed three-button mouse. Consequently, when employing a mouse with one or two buttons, it is important to understand the mouse-button equivalencies MOE employs.

Note that a scroll wheel often also operates as a clickable middle button.

The following table indicates MOE mouse-button equivalencies:

3-Button Mouse 2-Button Mouse
Left button Left button
Middle button Right button
Right button Press and release the Alt key

Note! MOE automatically detects if a 2- or 3-button mouse is available. If this check fails, or one wants to force the 2-button mouse mode, the detection can be overridden in the Mouse Configuration panel.

In MOE, MOE | Help | Mouse opens the Mouse Reference panel, which is a quick reference for mouse operation in the main MOE windows.

macOS Mouse Configuration

macOS supports mice with two or three buttons. The default mouse configuration is X11 three-button mouse emulation. Emulation should be disabled when using an actual three button mouse as this may render some command combinations unavailable (see: X11 | Preferences | Input).

Mouse settings can be modified for use with an actual three-button mouse in System Preferences | Keyboard & Mouse | Mouse. Recommended settings are the primary button on the left side of the mouse, the secondary button on the right side and the middle button, or scroll wheel, as button 3.

With a one-button mouse one can use the trackball in the footer of the MOE Window to rotate, translate, and zoom the molecular system.

Popup Menus: The Mouse Popup Button

Popup menus are used in many MOE windows such as the Sequence Editor, Database Viewer and Text Editor. To display a popup menu, position the cursor over a specific object or area in the given window (e.g. a residue in the Sequence Editor or the text area in a Text Editor) and press and release the mouse popup button. The mouse popup button is dependent on the type of mouse being used:

3-Button Mouse 2-Button Mouse
Right mouse button Alt key

The Alt key will display the popup menus regardless of the mouse type; i.e. it works even with a 3-button mouse. The popup menu disappears when the mouse is clicked elsewhere in the window or Esc is pressed. Popups are described in the chapters dedicated to each of the MOE windows.

What if I don't have an Alt key?

Certain operations in MOE require pressing the Alt key, which is commonly used in conjunction with the mouse. On macOS, the Alt and Option keys are synonymous.

Using the Mouse in the MOE Window

Selecting Atoms

Note: Mouse selections are subject to the state of the R, S, and L buttons.

Action Using the Left Button
Select an atom Click on the atom. This will deselect all other atoms.
Select a bond (two atoms bonded together) Click on the bond joining the atoms. This will deselect all other atoms.
Select all atoms of the same class (ligand, receptor, solvent) in the same connected component as the atom Double-click on the atom
Toggle selection state of an atom Shift-click on the atom
Select all atoms of parent residue of an atom Ctrl-click on the atom
Toggle selection state of parent residue atoms Shift+Ctrl-click on the atom
Select a group of atoms Drag over atoms
Add more atoms to a group of selected atoms Shift-drag over atoms
Select a group of atoms AND atoms of parent residues Ctrl-drag over atoms
Toggle selection state of parent residue atoms Shift+Ctrl-drag over atoms
Select atoms near an atom, deselecting all other atoms Click on the atom, then click again and drag the selection sphere with the center of the first atom to select all atoms inside the sphere.
Toggle selection state of atoms near an atom Click on a selected atom and Shift-drag the selection sphere. If all atoms inside the sphere are selected then they will be deselected, otherwise the unselected atoms in the sphere will be selected as well.
Select atoms AND atoms of parent residues near an atom Click on the atom, then click again and Ctrl-drag the selection sphere.
Toggle selection state of parent residue atoms near an atom Click on a selected atom and Shift+Ctrl-drag the selection sphere.
Select atoms near multiple selected atoms Click on one of the selected atom and Alt+Shift-drag selection spheres over all selected atoms.
Clear selection of atoms Click away from any atoms

Selecting Ribbons

When the ribbon segment of a residue is selected or unselected with the mouse, all the atoms of that residue are selected/unselected as well.

Note: Mouse selections are subject to the state of the R, S, L, and ⇨ buttons. With the ⇨ button one can enable/disable the selectability of ribbons in general.

Operation Left Mouse Button
Select ribbon segment of a residue, deselecting all other ribbons Click on the ribbon segment
Select all ribbons in the same connected component as the picked ribbon Double-click on the ribbon
Toggle selection state of a ribbon Shift-click on the ribbon
Select a group of ribbons, deselecting all other ribbons/atoms Drag over ribbons
Add more ribbons to a group of selected ribbons Shift-drag over ribbons
Clear selection of ribbons Click away from any ribbons

Panning, Rotating and Zooming Atoms

Action Using the Middle Button
Pan view in x and y axes Shift-drag the middle button in rendering area
Rotate view Drag the middle button in rendering area
Rotate view about z axis only Drag the middle button anywhere in perimeter of rendering area
Zoom in/out Ctrl-drag the middle button in rendering area
Change the center of rotation Click the middle button on the atom that will be the new center of rotation. Rotate the system by dragging the middle mouse button.

Setting the center of rotation also sets the focal plane of the stereo display and focal blur effect.

To reset the center of rotation to the default, click the middle button away from any atoms. (By default, the center of rotation is the center of mass of the molecular system.)

Using the Scroll Wheel

Some mice come with a scroll wheel. In the MOE Window, this wheel can be used to zoom in and out.

Editing Atoms

When the Alt key is pressed, mouse movement modifies the positions of the selected atoms.

Action Middle Button in MOE Window
Translate selected atoms Shift+Alt-drag the middle button
Rotate selected atoms Alt-drag the middle button
Rotate selected atoms about view z axis only Alt-drag the middle button in perimeter of rendering window
Action Left Button in MOE Window
Rotate about a bond (dihedral angle) Alt-drag the left button

Note: Requires that exactly two bonded atoms be selected. The side that moves is the one with fewer atoms or the one with no fixed atoms. Has no effect on ring bonds.

If the Alt Key is Bound to Window Movement

On some systems, it is possible that Alt will not work as described, e.g. instead it has been mapped to window movement. This can be corrected by editing the Window Preferences manager:

  • in Linux using GNOME desktop environment

    The Window Preferences panel can be opened using the command line (the terminal) by the user that launches MOE.

        $ gnome-window-properties
    With the Window Preferences panel open, re-map the mouse "Movement Key."
    1. Select Super (or "Windows logo") or Ctrl
    2. Click Close

    The Alt key should now react as described in the MOE manual and the Super (or "Windows logo") or Ctrl key can now be used to resize, or move windows.

  • in Linux using Desktop Environment (KDE)

    The preference panel is accessible from the KDE Control Center. Go to the Desktop/Window Behavior panel and select the "Windows Actions" tab. In the "Inner Window, Titlebar & Frame" area, change the "Modifier Key" option from Alt to Meta or Ctrl.

  • in the Common Desktop Environment (CDE)

    MOE's rotate-bond uses Mod1-Drag1. Mod1 is usually bound by XWindows to Alt and/or Meta (both Alt_L and Alt_R). Unfortunately, the CDE window manager dtwm binds Alt-Drag1 to window move. (They call the event Alt<Btn1Down>). To undo it follow these steps:

    • Edit your "dtwmrc" configuration file, typically located in one of 
          $HOME/.dt/$LANG/dtwmrc
    $HOME/.dt/dtwmrc

      If you do not have a dtwmrc file then you can copy (into $HOME/.dt/dtwmrc) the default file typically located in one of the following places:

          /etc/dt/config/$LANG/sys.dtwmrc
    /etc/dt/config/sys.dtwmrc
    /usr/dt/config/$LANG/sys.dtwmrc
    /usr/dt/config/sys.dtwmrc
    • Delete or comment-out any lines that define Alt<Btn1Down> binding in the Buttons DtButtonBinding class. Look for the paragraph 
          Buttons DtButtonBindings
    {
        ...
        Alt<Btn1Down> icon|window             f.move ...
        ...
    }

      and comment out the line:

          #   Alt<Btn1Down> icon|window             f.move
    • Restart the window manager by choosing the entry named "Restart..." or "Restart Workspace Manager..." (or something similar) in the root or desktop popup menu (right-click on the desktop).

If Shift+Alt Switches between Input Languages

By default, multilingual Windows machines bind Shift+Alt to the command of switching between input languages, which changes the keyboard mapping. MOE users should disable the Shift+Alt shortcut and either switch languages explicitly in the language bar, or assign different shortcuts (e.g. Shift+Alt+1 and Shift+Alt+2) to individual languages.

  • On Windows 7, the key sequence for changing input languages can be specified in:
    Control Panel | Region and Language | Keyboards and Languages | Language Bar | Advanced Key Settings
  • On Windows 8, the key sequence for changing input languages can be specified in:
    Control Panel | Language | Advanced Settings | Change language bar hot keys

Using the Mouse in the Sequence Editor

Selecting Chains

Action Using the Left Button
Select a chain Click on the chain number. This will deselect all other chains.
Select a tag Click on the tag name. This will select all chains grouped under the same tag, including all subunits, and deselect all other chains.
Toggle selection state of a chain or tag Ctrl-click on the chain number or tag name.
Select a range of chains or tags Click on first chain or tag and Shift-click on last chain or tag. This will fill in the range from the previous selection of the same type.
Select all chains or tags Click on Tag or Chain (tag/chain button). Chain buttons will select all chains in the current subunit whereas the Tag button will select all subunits.
Note: If all chains were already selected, this will deselect all chains.
Clear selection of chains or tags Click in the empty space below the chains or tags.

Selecting Residues

Action Using the Left Button
Select a residue Click on the residue. This will deselect all other residues.
Toggle selection state of a residue Ctrl-click on the residue.
Select a range of residues within the same chain Click on first residue and Shift-click on last residue.
Select a column of residues, i.e. residues that occupy the same alignment position across chains Click on the position in the alignment ruler.
Toggle selection state of a column of residues Ctrl-click in alignment ruler.

Note: If any residue in the column was previously unselected, this will select all residues. However, if all residues in the column were already selected, this will deselect all residues.
Select a range of residue columns Click in ruler over first column and Shift-click in ruler over last column.
Select a group of residues Drag over desired residues.
Note: selection indicated in purple text in footer.
Add more residues to a group of selected residues Ctrl-drag over desired residues.
Note: selection indicated with plus (+) sign in footer.
Mask selection of a group of residues Shift-drag over desired residues in mask. This will select only those residues both in mask selection and which were previously selected.
Note: mask selection indicated with ampersand (&) in footer (logical AND).
Remove residues from a group of selected residues Alt-drag over desired residues.
Note: selection indicated with minus (-) sign in footer.
Clear selection of residues Click away from residues, chains or ruler.

Moving Residues

Action Using the Middle Button
Move a residue (gaps not preserved) Drag the middle button over the residue. A red border highlights the residue being moved.

Note: This moves the residue right or left, changing its alignment position. Any residues encountered will be pushed along, like beads on a string. Gaps between residues are not preserved.
Move several residues (gaps not preserved) Select the residues to move (residues can belong to more than one chain and do not have to be consecutive), and press Alt while dragging the middle button.

Note: This moves selected residues right or left, like beads on a string. Selected residues maintain their relative alignment positions while moving.

Tip. One can also move the selected residues with the Left and Right arrow keys on the keyboard.
Move a residue (preserve gaps) Shift-drag the middle button over the residue.

Note: Unlike the beads-on-a-string method, this pushes all residues while maintaining gaps.
Slide an entire chain Ctrl-drag the middle button over the residue.
Move residues in multiple chains (preserve gaps) Select the chains in the Chains or Tag column and then Shift-drag the middle button over a residue in the selected chains.If a residue of an unselected chain is chosen, only that chain will be moved.

Note: Unlike the beads-on-a-string method, this pushes all residues while maintaining gaps.

Tip. Alternatively select one residue in the selected chains and use the Shift-Left and Shift-Right arrow keys on the keyboard.
Slide multiple chains Select the chains in the Chains or Tag column and then Ctrl-drag the middle button over a residue in the selected chains. If a residue of an unselected chain is chosen, only that chain will be moved.

Tip. Alternatively select one residue in the selected chains and use the Ctrl-Left and Ctrl-Right arrow keys on the keyboard.

Note: Residues cannot be moved beyond the start of the alignment ruler.

Mouse Configuration Panel

MOE allows the reconfiguration of the MOE window mouse behavior. This is useful, if e.g. only a two button mouse is available or the user wants to mimic a mouse mapping of another program. To change the mouse behavior, open the Mouse Configuration panel using MOE | Settings | Mouse. This panel also allows to assign custom SVL code to mouse button clicks.

Mouse Reference Panel

The Mouse Reference panel is opened using MOE | Help | Mouse, and is intended as a quick reference for mouse operation in the MOE Window, Sequence Editor, and Database Viewer windows.

The panel pages are organized by window, accessible using the buttons at the top of the panel. The window-dependent categories of operation are viewed using the buttons at the bottom of the panel. The mouse key to press is indicated in blue, with mouse drag shown as blue arrows.
The reference of the MOE Window page is dynamic and changes if the mouse configuration is altered with the Mouse Configuration panel. But only changes to MOE default mouse actions are shown, not additionally added actions. Changed mouse actions are marked red.

MOE Footer and Trackball

Note that many MOE Window mouse operations can also be performed using controls in the MOE Footer area, notably the Trackball.

SpaceNavigator®

MOE supports 3D motion controllers and 6-degrees-of-freedom (6dof) devices from 3Dconnexion such as the SpaceNavigator®. Use of such devices requires installing the 3Dconnexion driver software. On macOS, additionally, the 3DxMagellanProxy application (available from 3Dconnexion) needs to be installed and running before MOE is started.

When connected, the 6dof cap controller of the SpaceNavigator® controls the translation and rotation of the molecules in the MOE Window in all directions. If the Alt key is pressed, only the currently selected atoms will be moved.

The buttons of the 3Dconnexion devices can be programmed using the driver software and can be set up to execute SVL commands. To do this, the buttons must first be configured in the driver to send mouse button events to MOE. The SVL command to be executed is associated with a particular button by setting the appropriate RC variable. For example:

SetRC ['spacemouse.button1', 'ViewCenter[]']

will configure button 1 of the SpaceNavigator® to execute the ViewCenter command and recenter the molecules inside the MOE Window. Keyboard modifiers can be used in conjunction with the button assignments, and are indicated in the RC variable by the corresponding suffix: s=Shift, c=Control, a=Alt. For example, the RC variable spacemouse.button3.s will configure the third SpaceNavigator® button press while the Shift key is pressed.
