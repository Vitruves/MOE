Graphics Object Manager

The Graphics Object Manager lists the graphical entities currently created in the MOE system. Typical graphics objects are protein patch surfaces, domain motif vectors, and user-built objects (constructed using SVL graphics functions). The Graphics Object Manager can be used to hide, show, delete and rename such objects.

Important! The Graphics Object Manager does not control the display of atoms, bonds, meters, the coordinate axis, the periodic box, or backbone ribbons. It also does not control the display of "quick" surfaces created using MOE | RHS | Surface or using the System Manager, or of surfaces created using the Surfaces and Maps panel.

To open the Graphics Object Manager, select Window | Graphics Objects from either the MOE Window, the Sequence Editor or a Database Viewer. The following panel appears:

The list updates dynamically and indicates whether current objects are shown or hidden.

Tip   A shortcut to opening the Graphics Object Manager is to use the keystroke combination Ctrl–g while the focus is in one of the MOE, Sequence Editor or Database Viewer windows.

One or more graphics objects can be selected. To select multiple objects, use Ctrl–click or Shift–click to add to the current selection set. Select one or more objects followed by the action to perform.

Tip   Double-clicking on a selected entry in the list will show the selected object and hide all other objects, providing a quick means of singling out an object of interest.

Hide The selected graphics objects are no longer drawn in the MOE Window. The word [Hidden] appears beside the entries in the panel.
Show The selected graphics objects are drawn in the MOE Window. The word [Shown] appears beside the entries in the panel.
Toggle Toggles the visibility of the selected graphics objects. A hidden object becomes shown, and a shown object becomes hidden.
Delete Deletes the selected graphics objects.
Name The name of the last selected graphics object is displayed in the Name text field below the list. To change the name of the selected object(s), type the new name into the text field and press <Return> or Apply.
Transparency The transparency levels of the last selected graphics object are displayed in the Transparency subpanel. When the Faces option is set to Front And Back, the transparency of the front and back faces of the surface can be adjusted individually. When the option set to Identical, the front face and the back face transparency levels are set to the same value and modified simultaneously. To set the transparency level to the default, press 0. Positive values result in greater transparency. Any change will be applied to the entire selection.
Material The material properties of the last selected graphics object are displayed in the Material subpanel. When the Faces option is set to Front And Back, the material properties of the front and back faces can be adjusted individually. When the option is set to Identical, the front face and the back face material values are set to the same value and modified simultaneously. Values for Shading, Gloss and Brightness (S, G and B) can be entered manually (press Apply for the changes to take effect), or various preset materials can be chosen from the drop-down menu. (These controls are explained in Visualization Setup.) Any change will be applied to the entire selection.

Note: MOE treats graphics objects differently from molecular objects. For instance, when a file is closed using File | Close, MOE displays a dialog in which the option of deleting graphics objects is enabled by default. If the option is turned off, graphics objects remain in the system although the molecular system is deleted.
