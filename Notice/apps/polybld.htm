This chapter describes how to specify repeat units and to combine them to create polymeric molecules.

Building Polymers

The Polymer Builder is used to build polymer molecules composed of either a single repeat unit (homopolymers) or two or more repeat units (copolymers).

To open the Polymer Builder, choose MOE | Edit | Build | Polymer. The following panel appears:

Polymer Builder Panel Reference

Polymer Name Set the name of the resulting polymer chain. Residue names in the polymer chain will be set to the first three letters of the repeat unit chain names, or the first three letters of the result chain name if the former is null.
Type Choose how to combine the repeat units. Homo builds a homopolymer from a single repeat unit, Random builds a co- or multipolymer by randomly selecting from the input repeat units, Alternating will alternate between all input repeat units, and Block will create a block of n repeats of each unit in turn, where n is specified in the Multiplicity column. Only the first repeat unit menu will be active in Homo mode. In Alternating mode, only the first Multiplicity entry will be active, and determines the number of repeats of each unit.
Preserve Multimeric Repeat Units If enabled, then the repeat unit is replicated as-is. If disabled, the repeat unit is converted into a monomer by reparenting the atoms of all residues in the repeat unit to the first residue. Note that if the repeat unit is retain in its multimeric form, the residue ordering may not follow a linear sequence in the resulting polymer chain, depending on where the tail and head connection points are. This option only has effect when the repeat unit comprises multiple residues.
Dihedral Mode Set the backbone angles of the polymer. In Angle mode, each bond between repeat units is set to the specified angle. In Helix mode, all bonds between backbone atoms are set to the specified angle. In Random mode, each bond between repeat units is set to a random angle.
Repeat Units Select which repeat units are to be combined in the final polymer, the number of times each unit will appear (Multiplicity) and their Tacticity, which is the relative arrangement of the repeat unit sidechains with respect to polymer backbone. Chains will be listed as a potential repeat unit if they contain a single connected molecule which has head and tail atoms defined, i.e. hydrogen atoms named A1 and A0 respectively. Setting a tacticity of 0 (zero) means that all the sidechains will be on the same side of the polymer backbone (isotactic). Setting tacticity to 1 will create a syndiotactic polymer, where the sidechains alternate from one side of the backbone to the other along the chain. Non-integral tacticity values are interpreted as the probability of the sidechain appearing on a particular side of the backbone.
Monomer Database A database of 253 standard monomeric repeat units is supplied with MOE in the file $MOE/lib/polymer.mdb. Clicking Open will open this file in a Database Viewer, and Close will close it again.
Show Entries Use this bank of checkboxes to restrict the visible entries of the Monomer Database in the Database Viewer to those that contain a particular chemical group. If multiple checkboxes are enabled, then all the specified groups must be contained in the repeat unit. If the desired monomer is not available in the database, a new monomer can be defined using the Define Repeat Unit tool, as described below.

Press the OK button to create the polymer.

Creating New Repeat Units

If the desired repeat unit is not already defined in MOE's database of standard repeat units (contained in $MOE/lib/polymer.mdb), a molecule can be marked as a repeat unit for use with the Polymer builder using the Define Repeat Unit application. To launch the application, choose MOE | Edit | Build | Repeat Unit.

To define a new repeat unit, first build a molecule in MOE using the Builder, for example. Select a hydrogen atom and click Define Tail to set the tail of the repeat unit, select a second hydrogen atom and click Define Head to set the head, then select one of the heavy atoms that link the head and tail and click the Set Backbone button. The name of the repeat unit can be specified by entering it into the Name text box and clicking Apply. The repeat unit can now be used with the Polymer Builder application.

Note that for multimeric repeat units, the tail atom (A0) is best defined in the first residue of the repeat unit, and the head atom (A1) in the last residue. This is to ensure the sequential nature of the resulting polymer chain.

Example: Building polyvinyl chloride

Polyvinyl chloride (PVC) is formed by a free-radical mechanism, and as such it is an atactic polymer with an amorphous structure. To build a PVC molecule using the Polymer Builder in MOE:

  1. Launch the Polymer Builder.

    MOE | Edit | Build | Polymer

  2. Select the repeat unit.
    In the Polymer Builder panel:
    1. Press Open to open the database of standard repeat units $MOE/lib/polymer.mdb into a Database Viewer.
    2. Turn on the Halide box. In the Database Viewer, only those repeat units that contain halogen atoms will be shown.
    3. The polyvinyl chloride unit is entry 209 and has the name "poly(vinyl chloride)". Load this entry into MOE:

      DBV | Molecule Cell | Popup | Send to MOE.

    4. In the Polymer Builder panel, set the repeat unit name by entering polyvinyl chloride into the Polymer Name box.
  3. Set the polymer properties.
    1. Choose Random from the Dihedral Mode menu.
    2. Set the Multiplicity to 20 and the Tacticity to 0.5.
  4. Build the polymer.
    Press OK in the Polymer Builder panel. Use MOE | Render | Center to center the resulting PVC polymer in the MOE Window, then press MOE | RHS | Minimize to obtain the following result.
