Overview

The Capture application provides ways to easily keep track of different system states. Typical uses of Capture include journaling work, recording thoughts and annotations, storing manually modified ligands with associated scores and remarks, and navigating to different saved views of interest in a system. In this context, a capture is an association of a complete system state, a title, a set of comments, ligand properties, and a screenshot of the MOE window, with optionally the system view setup. These captures are stored in user-created Capture files, which are specialized MOE databases and have .mdb as their file extension.

Capture menu

The Capture menu is reachable with MOE | RHS | Capture. The same menu is available from MOE | File | Capture; in the remainder of this document, all menu references will be to MOE | RHS | Capture, with the understanding that they can be equivalently found under the MOE | File | Capture menu. Through this menu, new captures can be added, Capture files can be opened and closed, and, if a Capture file is currently open, a list of the captures contained in the file can be accessed.

Open Capture File

List of captures

To start using the Capture application, a Capture file must first be opened through MOE | RHS | Capture | Open. This brings up the Open Capture File panel. To open an existing Capture file, select Open, then either use the browse button to navigate to the file or enter its path in the text field, then press OK. Alternatively, double-click in the list of recently-used Capture files as displayed in the Recent list; the file will open directly.

To create and open a new Capture file, press Create, then either use the browse button to navigate to the directory in which to save the file, then enter the name of the file, or enter the full path and name of the new file in the text field. Once the name has been specified, press OK. The file suffix (.mdb) is automatically added if omitted. If no path is given, the file will be created in the current working directory. If the file name corresponds to an existing file, that file will be overwritten. Once a Capture file is opened, it will remain open until it is closed by using the Close menu option or by closing the MOE session.

New captures can be added to a Capture file using MOE | RHS | Capture | Add. The captures saved in the currently-open Capture file can be accessed from a popup list panel raised by MOE | RHS | Capture | List; note that this menu item is available only when a Capture file is currently open.

The scrollable and sortable list of captures displays titles and creation dates, along with ligand properties, if any. Clicking on an entry will close the current system and load into MOE the system saved in that entry.

Warning! Loading a capture will close the current MOE system without putting up a confirmation dialog.

If the current Capture file contains five or more captures, a search bar will appear, providing the ability to filter the list using search patterns.

The index number displayed in the first column of the list corresponds to the position of that capture's entry in the Capture file. When a capture is loaded, its entry index is displayed in a message in the upper-right corner of the MOE Window. The index is also displayed in the title bar of the Edit panel to help identify which entry is being edited. In the Capture footer, the indices are used to number the browsing buttons.

More features such as capture navigation, reordering, and update are available in the Capture footer and panel.

Adding a Capture

The Add panel is used to capture the current MOE system. It is opened with MOE | RHS | Capture | Add. A visual snapshot of the system to be saved is displayed. If MOE | RHS | Ligand | Ligand Properties is currently enabled, ligand properties will be included in the display. The Title, Comment and Save options can be set as desired. Pressing OK will add a new capture to the current Capture file.

The cursor position can be switched between the Title and Comment text fields by pressing Tab on the keyboard. A pulldown list located in the rightmost position of the Title text field contains title suggestions based on the names of the chains in the system and previous captures. Pressing OK with empty Title and Comment text fields will create a capture with the default title that consists of the names of the active chains; by default the comment is left empty. Note: use Shift-Tab in the Comment field to add a tab in the text.

The Save options are preserved from session to session.

Active Tags If on, only the tags with active atoms will be saved in the new capture, otherwise all chains will be saved.
Surfaces If on, surfaces will be saved in the new capture, otherwise surfaces will not be saved.
Ligand Only If on, only the active ligand will be saved in the new capture, otherwise the whole system will be saved (in accordance with the Active Tags setting).
Theme If on, the current View Setup will be saved as part of the new capture. Note that if Theme is disabled, when the capture is subsequently loaded, whatever theme is in use at that time will continue to be in effect, regardless of what theme is displayed in the bubble help thumbnail in the list of captures or in the Add or Edit panels.

Capture Footer and Panel

Capture can be launched as an attached or detached panel from the list of captures (MOE | RHS | Capture | List, only available when a Capture file has been opened). In attached mode, Capture opens in the MOE Footer, otherwise it can be opened as a detached panel. Pressing the top right button in the list popup displayed by the menu option List... will bring up the Capture footer or detached window, depending on which preferred location is chosen in the Capture Settings. The attach button opens the footer mode and the detach button opens the detached window. The default mode is attached at the bottom of the MOE window.

Capture footer

Capture Footer List Panel

The Capture footer is meant to provide easy access while being unobtrusive, making it suitable for presentations and working with multiple simultaneous panels. It allows one to:

  • Preview and load any capture from the current Capture file using the numbered list, along with navigation buttons.
  • Browse, search, and manage the list of captures with the footer list panel opened by the List button.
  • Open the Add panel to add a new capture with the Add button.
  • Open the Edit panel to edit the currently selected capture with the Edit button.

The index numbers in the numbered list of the captures correspond to the positions of the entries in the Capture file. Hovering the mouse over a numbered button will display a preview of the capture with its title. Pressing a numbered button or the navigation buttons will close the current system and load the corresponding capture, highlighting it in the list.

The footer list popup is similar to that opened by MOE | RHS | Capture | List, but has more features. The name of the current Capture file is displayed at the top of the popup and can be changed by selecting Open in the menu . This menu also provides an Open DBV option for opening the Capture file in a Database Viewer.

The capture list displays the capture indices, titles, and creation dates, plus ligand properties if any. It is scrollable and sortable. Clicking on an entry will close the current system and load the system saved in that entry.

Warning! Loading a capture will close the current MOE system without putting up a confirmation dialog.

Typing in the search bar will dynamically search the text of all titles, comments, and properties shown in the list for matches to the search pattern as it is being entered. This filtering is case insensitive.

The footer list panel also contains additional controls. Similarly to those in the footer itself, the Add and Edit buttons will open their respective panels. The move up and move down buttons can be used to change the position of the selected capture in the list. Doing so will change the position of the capture's entry in the Capture file. Note: changing position is only available when the capture list is sorted by index and unfiltered. Pressing the delete button will remove the selected capture from the Capture file.

Warning! Deleting a capture from the Capture file cannot be undone.

Pressing the detach button in the footer will close the footer and open the detached Capture window. This window is identical in function to the footer popup, but additionally displays the title and complete comment of the selected capture.

Detached Capture panel

Capture Modification

Saved captures can be updated using the Edit panel. This panel allows one to modify the title and comment, and to optionally replace the complete system of a previously saved capture.

The system saved in the current capture and the current MOE system are displayed side-by-side in the panel. Selecting Keep Saved System will leave the system saved in the capture unmodified, while selecting Overwrite With Current System will overwrite the saved system with the current system in MOE.

The other options of the Edit panel are the same as in the Add panel. The Title and Comment text fields contain the title and comments saved in the selected capture and can be edited. As in the Add panel, the cursor position can be switched between the text fields by pressing Tab. A pulldown list located in the rightmost position of the Title text field contains title suggestions. Note that the Save options are only available when Overwrite With Current System is selected,

Pressing OK will update the capture. Warning! This operation cannot be undone.

Capture Settings

The gear button opens a panel to configure the default settings for Capture.

Open Mode Choose the default mode for Capture when the Capture panel is launched from the list of captures opened using MOE | RHS | Capture | List.
  • Detached. Capture will open in a separate window. In the list of captures, the launch button will appear as .
  • Attached. Capture will open in the MOE Footer. In the list of captures, the launch button will appear as .
Transitions Enable animated transitions between captures and set the transition duration, in seconds.
Bubble Help Enable a picture of the system or the capture's comment to be shown when the mouse is hovered over a capture in the list.
Close Capture File If on, closing the system will also close the current Capture file.
