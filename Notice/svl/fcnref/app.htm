Function Extension: Apply-to-Each (app, apt)

Syntax

app function args
apt function args

Description

app function args

app applies a function to each of the top-level elements in the argument vector args. It can be construed as an implicit form of looping.

svl> add [ [1,2,3], [10,20,30] ]
[11,22,33]
svl> app add [ [1,2,3], [10,20,30] ]
[6,60]

apt function args

A special form of app is apt, short for app tr, where tr is the transpose function. apt first matrix-transposes the arguments of the function to be applied, and then proceeds with the app function.

svl> apt reverse [ [1,2,3], [4,5,6] ]
[ [4,1], [5,2], [6,3] ]
