Vector Type Identifiers: isflat, ...

Syntax

    val = isflat vector
    val = isflatchar vector
    val = isflatnum vector
    val = isflattok vector

    val = isscalar vector
    val = isscalarchar vector
    val = isscalarnum vector
    val = isscalartok vector

Description

Function Description
isflat True if each element of the argument vector is a scalar and all elements are of the same type.
isflatchar True if each element of the argument vector is a scalar and all elements are of type 'char'.
isflatnum True if each element of the argument vector is a scalar and all elements are of type 'num'.
isflattok True if each element of the argument vector is a scalar and all elements are of type 'tok'.
isscalar True if the argument is a scalar (single number, token or character).
isscalarchar True if the argument is a scalar and of type 'char'.
isscalarnum True if the argument is a scalar and of type 'num'.
isscalartok True if the argument is a scalar and of type 'tok'.
