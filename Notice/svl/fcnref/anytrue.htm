SVL Functions: anytrue, alltrue, anyfalse, allfalse

Syntax

anytrue v
alltrue v
anyfalse v
allfalse v

Description

These functions search the set of (arbitrarily nested) scalars that are contained in the vector v and return either 0 or 1, depending on the presence of scalar values 0 (zero), '' (empty token), or other than 0 or ''.

anytrue returns 1 if and only if vector v contains at least one (arbitrarily nested) scalar other than 0 or ''.
alltrue returns 1 if and only if vector v does not contain any scalar of value 0 or ''.
anyfalse returns 1 if and only if vector v contains at least one scalar of value either 0 or ''.
allfalse returns 1 if and only if vector v does not contain any scalar of value other than 0 or ''.

Example:

svl> v = 0
svl> [anytrue v, alltrue v, anyfalse v, allfalse v]
[0,0,1,1]

svl> v = []
svl> [anytrue v, alltrue v, anyfalse v, allfalse v]
[0,1,0,1]

svl> v = [[0,1], 'a']
svl> [anytrue v, alltrue v, anyfalse v, allfalse v]
[1,0,1,0]

svl> v = 5
svl> [anytrue v, alltrue v, anyfalse v, allfalse v]
[1,1,0,0]
