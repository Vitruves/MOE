Rotate Functions: rot...

Syntax

    rotated_v = rot [v,num]

    rotated_left_v = rotl v
    rotated_right_v = rotr v

    rot_left_poked_v = rotlpoke [ v, val ]
    rot_right_poked_v = rotrpoke [ v, val ]

Description

rotated_v = rot [ v, num ]

Rotates source vector v by num positions. Positive num rotates right, negative rotates left.

    svl> rot [ [1,2,3], 1 ] // rotate right
    [3,1,2]

    svl> rot [ [1,2,3], -1 ] // rotate left
    [2,3,1]

rotated_left_v = rotl v
rotated_right_v = rotr v

Left and right rotate by one position. Note that:

    rotl a === rot [a, -1]
    rotr a === rot [a, 1]

    svl> rotl [1,2,3]
    [2,3,1]

    svl> rotr [1,2,3]
    [3,1,2]

rot_left_poked_v = rotlpoke [ v, val ]
rot_right_poked_v = rotrpoke [ v, val ]

Rotate left or right by one position, replacing the element that is carried from one end of the vector to the other by val. Note that:

    rotlpoke a === poke [ rotl a, len a, val ]
    rotlpoke a === rotl poke [ a, 1, val ]

    svl> rotlpoke [ "sword", "y" ]
    "wordy"

    svl> rotrpoke [ [1,2,3,4,5], 99 ]
    [99,1,2,3,4]
