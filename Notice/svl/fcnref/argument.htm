Function Argument

Syntax

argument []
argval = argument v

Description

When passed null, returns the argument vector of the current function. When passed a flat numeric vector v, returns

peek [ argument[], v ]

argument is useful for accessing arguments to script files, and for processing optional arguments.

Here is an example of how to extract optional arguments passed to a function:

function f [a,b,c]
    local optargs = drop [argument[], 3]; // drop a, b, c

    ...

endfunction

Advanced Topic

The argument of the boot function is the task argument. Thus:

svl> task_eval [ 'argument[1,2]', [ [10,20], 30 ] ]
[20, '']
