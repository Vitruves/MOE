Permutation Functions: perm, x_perm, parity

Syntax

    permuted_v = perm [ v, idx_list ]
    perm_indices = x_perm idx_list
    flag = parity v

Description

permuted_v = perm [v,idx_list]

Permutes the source vector according to the index list. The index list gives the destination position of each source element. perm is similar to put except that the result is formed from elements of only one vector. In addition, perm requires that the index be a valid permutation, i.e. each element is used exactly once.

Note that:

    get [perm [v, idx_list], idx_list] === v

For example:

    svl> perm ["cyrano", [1,4,2,3,6,5]]
    "crayon"

    svl> put [[], [1,4,2,3,6,5], "cyrano"]
    "crayon"

perm_indices = x_perm idx_list

Takes as argument the permutation vector idx_list and returns a new index list perm_indices, such that:

    perm [v, idx_list] === get [v, perm_indices]

Note that:

    x_perm idx === perm [x_id idx, idx]

    svl> get ["cyrano", x_perm [1,4,2,3,6,5]]
    "crayon"

flag = parity v

Calculates the parity of the permutation (x_id v : rank v). The parity of a permutation is the parity of the number of transpositions that form the permutation.

    svl> parity "abc"
    0

    svl> parity "acb"
    1

    svl> parity "cba"
    1
