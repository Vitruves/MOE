Exponentiation Functions

Syntax

y = cbrt  x
y = cube  x
y = exp   x
y = expm1 x
y = log   x
y = log10 x
y = log1p x
y = log2  x
y = logb  [x, b]
y = pow   [x, p]
y = sqr   x
y = sqrt  x

y = logaddexp [x1,...,xN]

Description

SVL provides a number of exponentiation functions that are used to raise values to specific powers. A collection of inverse functions are also provided.

Function Value Description
cbrt x The cube root of x.
cube x The cube of x.
exp x The value e raised to x.
expm1 x The value e raised to x less one. Use this function for improved precision when x is near zero (compared to exp).
log x The natural logarithm of x. NaN is returned for x<0.
log10 x The base 10 logarithm of x. NaN is returned for x<0.
log1p x The natural logarithm of (x+1). NaN is returned for x<-1. This function provides improved precision for x close to zero (compared to log).
logaddexp [x1,...,xN] The natural logarithm of the sum of exponentiated values. This function is appropriate for situations for summing numbers of significantly disparate orders represented in log-space.
logb [x,b] The base b logarithm of x. NaN is returned for x<0 or b non-positive.
log2 x The base 2 logarithm of x. NaN is returned for x<0.
pow [x,p] The argument x raised to the power p. NaN is returned if p is not an integer and x<0.
sqr x The square of x.
sqrt x The square root of x. NaN is returned for x<0.

The unary operators (like sqr and cube) apply to all scalar values in the argument vector (no matter how deeply nested). The binary operators (like pow and log1p) match elements of the arguments in pair no matter how deeply nested. Unit extension applies.
