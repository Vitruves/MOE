Sorting Functions: sam, sac

Syntax

    [i,m] = sam v
    [i,c] = sac v

Description

The function names are acronyms for sort-argument and mask and sort-argument and counts, respectively. These functions return a list of indices of elements in the sorted order (the sort "argument"), i, and a mask of unique elements, m, or the counts of segments of duplicate elements, c, respectively:

    i === x_sort v
    m === m_uniq sort v
    c === mtoc m

In other words,

    sort v == get [v,i]
    uniq sort v == (sort v) | m

For example:

    svl> v = "Bookkeeper"
    svl> [i,m] = sam v; print i; print m;
    [1,6,7,9,4,5,2,3,8,10]
    [1,1,0,0,1,0,1,0,1,1]

    svl> v[i] // sort v
    "Beeekkoopr"

    svl> v[i|m] // uniq sort v
    "Bekopr"

    svl> split [v[i], c] // sets of unique elements
    [ "B", "eee", "kk", "oo", "p", "r" ]

    svl> perm [pscan m, i]    // "consecutive" rank of v
    [1,4,4,3,3,2,2,5,2,6]

    svl> frank v              // "cumulative" rank of v
    [1,7,7,5,5,2,2,9,2,10]
