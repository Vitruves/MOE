Trigonometric Functions

Syntax

    v = acos  x
    v = acosh x
    v = asin  x
    v = asinh x
    v = atan  x
    v = atanh x
    v = atan2 [y,x]
    v = cos   x
    v = cosh  x
    v = norm  [x1,...,xN]
    v = normR [x1,...,xN]
    v = sin   x
    v = sinh  x
    v = tan   x
    v = tanh  x

Description

SVL provides a number of trigonometric functions.

Function Value Description
acos x arccos(x) The arccosine (inverse cosine) of x. The return value is in radians in the range [0,PI]. NaN is returned if x is not in the range [-1,1].
acosh x arccosh(x) The inverse hyperbolic cosine of x. The return value is in radians in the range [0,PI]. NaN is returned if x<1.
asin x arcsin(x) The arcsine (inverse sine) of x. The return value is in radians in the range [-PI/2,PI/2]. NaN is returned if x is not in the range [-1,1].
asinh x arcsinh(x) The inverse hyperbolic sine of x. NaN is returned if x is not greater than 1.
atan x arctan(x) The arctangent (inverse tangent) of x. The return value is in radians in the range [-PI,PI].
atanh x arctanh(x) The inverse hyperbolic tangent of x. NaN is returned if x is not in the range [-1,1].
atan2 [y,x] arctan(y/x) The arctangent (inverse tangent) of the quotient y/x in which the signs of x and y are used to determine the correct quadrant. Unlike atan(y/x), the function atan2[y,x] avoids the singularity when y=0. When x=y=0, the function returns 0. The return value is in radians in the range [-PI,PI].
cos x cos(x) The cosine of x.
cosh x cosh(x) The hyperbolic cosine of x.
norm [x1,...,xN] || (x1,...xN) || The Euclidean norm of the vector (x1,...xN). This is equivalent to the expression sqrt add sqr x.
normR [x1,...,xN] || (x1,...xN) ||F The Frobenius norm of the vector (x1,...xN). This is equivalent to the expression norm cat x.
sin x sin(x) The sine of x.
sinh x sinh(x) The hyperbolic sine of x.
tan x tan(x) The tangent of x.
tanh x tanh(x) The hyperbolic tangent of x.

The unary operators (like cos and sin) apply to all scalar values in the argument vector (no matter how deeply nested). The binary operators (like atan2) match elements of the arguments in pairs no matter how deeply nested. Unit extension applies.
