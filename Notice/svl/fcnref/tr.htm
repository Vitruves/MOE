Transpose Function: tr

Syntax

    transposed_a = tr a

Description

Matrix transposition: rows become columns, columns become rows. Useful for assembling the arguments for individual function calls from lists of arguments, and also for assembling lists of data from data records.

Unit extension applies to the top-level elements of unit length.

    svl> tr [ [1,2,3], [4,5,6] ]
    [ [1,4], [2,5], [3,6] ]

    svl> tr [ ['a','b','c'], [1,2,3],["x","y","z"] ]
    [ [ 'a', 1, "x" ], [ 'b', 2, "y" ], [ 'c', 3, "z" ] ]

    svl> tr [ "abcd","E","fghi"]
    ["aEf","bEg","cEh","dEi"]

    svl> tr []
    [ [] ]
