Set Difference Functions: diff, m_diff, ...

Syntax

values_only_in_v = diff [v,w]

diff_mask = m_diff [v,w]
diff_indices = x_diff [v,w]

values_not_in_both = sdiff [v,w]

Description

values_only_in_v = diff [v,w]

Returns all elements of v that do not occur in w, preserving ordering and duplicates in v.

svl> diff [ "afg", "cfigg" ]
"a"
svl> diff [ "cfigg", "afg" ]
"ci"

diff_mask = m_diff [v,w]

Returns a mask such that: diff [v,w] === v|msk

svl> m_diff [ "cfigg", "afg" ]
[1,0,1,0,0]

diff_indices = x_diff [v,w]

Returns an index list such that: diff [v,w] === v[idx]

svl> x_diff [ "cfigg", "afg" ]
[1,3]

values_not_in_both = sdiff [v,w]

Gives the symmetrical set difference. sdiff[v,w] returns cat[v,w] with all elements occurring in both v and w removed. The function preserves duplicates and the original ordering of its arguments.

svl> sdiff [ "afg", "cfigg" ]
"aci"
svl> sdiff [ "cfigg", "afg" ]
"cia"
