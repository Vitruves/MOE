Logical Character Identifiers: isalpha, ...

Syntax

    val = isalnum character_vector
    val = isalpha character_vector
    val = iscntrl character_vector
    val = isdigit character_vector
    val = isgraph character_vector
    val = islower character_vector
    val = isprint character_vector
    val = ispunct character_vector
    val = isspace character_vector
    val = isupper character_vector
    val = isxdigit character_vector

    val = isalnum token_vector
    val = isalpha token_vector
    val = iscntrl token_vector
    val = isdigit token_vector
    val = isgraph token_vector
    val = islower token_vector
    val = isprint token_vector
    val = ispunct token_vector
    val = isspace token_vector
    val = isupper token_vector
    val = isxdigit token_vector

Description

For each leaf character element of character_vector, the following functions return 1 for true, 0 for false.

For each leaf token element of token_vector, the following functions return 1 if ALL characters in the token are true, 0 if ANY of the characters in the token are false. 1 or 0 is returned defined by each function.

Function Description
isalnum True for any character for which isalpha or isdigit is true. 
    svl> isalnum "a3B. _+}"
    [1,1,1,0,0,0,0,0]
isalpha True for characters for which isupper or islower is true, i.e. for which none of iscntrl, isdigit, ispunct, or isspace is true. 
    svl> isalpha "a3B. _+}"
    [1,0,1,0,0,0,0,0]
iscntrl True for control characters. 
    svl> iscntrl "1x \n\t"
    [0,0,0,1,1]
isdigit True for any decimal-digit character (0-9). 
    svl> isdigit "a3B. _+}"
    [0,1,0,0,0,0,0,0]
isgraph True for any printable character, not including spaces. 
    svl> isgraph "a3B. _+}"
    [1,1,1,1,0,1,1,1]
islower True for lower-case letters, i.e. letters for which none of isupper, iscntrl, isdigit, ispunct, or isspace is true. 
    svl> islower "a3B. _+}"
    [1,0,0,0,0,0,0,0]
isprint True for any printable character, including spaces.
ispunct True for characters for which none of isalnum, isspace, or iscntrl is true. 
    svl> ispunct "a3B. _+}"
    [0,0,0,1,0,1,1,1]
isspace True for any white space, including newlines, vertical space, and for characters for which neither isalnum nor ispunct is true. 
    svl> isspace "a3B. _+}"
    [0,0,0,0,1,0,0,0]
isupper True for upper-case letters, i.e. letters for which none of islower, iscntrl, isdigit, ispunct, or isspace is true. 
    svl> isupper "a3B. _+}"
    [0,0,1,0,0,0,0,0]
isxdigit True for any hexadecimal-digit character (0-9, A-F, a-f). 
    svl> isxdigit "a3B. _+}"
    [1,1,1,0,0,0,0,0]
