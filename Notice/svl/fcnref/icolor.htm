icolor, AdjustColor

Syntax

    val     = icolor 'colorname'

    new_val = AdjustColor [val, amount]

Description

val = icolor 'colorname'

Returns the RGB encoded value corresponding to the named color, given as a token. The recognized colors are given in PLOT_COLORS. The function unit extends, i.e. it can be given a vector of tokens.

new_val = AdjustColor [val, amount]

Adjusts the color of an RGB encoded value and returns the new color (also RGB encoded). The adjustment is done in the CIE Lab color space. The argument amount is a value approximately in the range of -100 (dark) to 100 (bright).
