Plot Widget

Description

The Plot widget is a display area that is used to plot data and set plot-related attributes. Four different plot types are supported:

  1. function plot of one-dimensional data.
  2. histogram bar chart plot of one-dimensional data. Multiple data series can be plotted at the same time.
  3. scatter plot of two-dimensional data on X and Y axes. A third and fourth data dimension can be optionally visualized by coloring and sizing the data points according to the data values.
  4. contour plot of matrix data on a regular two-dimensional grid.

The function, histogram, and scatter plot support visualization of multiple data series at the same time.

Trigger

A selection box is drawn by dragging and releasing the left mouse button over the Plot area. Releasing the mouse button is the trigger.

Data Value

The selection box is returned as the data value. The plot widget returns [x1,y1,x2,y2], where [x1,y1] are the coordinates of the top left corner of the selection box and [x2,y2] are the coordinates of the bottom right corner of the selection box.

If the type attribute is 'modint' the key modifier information is returned as fifth value: [x1,y1,x2,y2,mod]. The key modifier bits include WMOD_CTRL, WMOD_SHIFT, and WMOD_ALT. They indicate if one of the modifier keys was pressed during the mouse selection. They can be checked with the following code:

    alt_pressed = bitand [mod, WMOD_ALT];

Position Dependent Bubble Help

By default one can specify only one common bubble help text over the whole plot widget. The attribute bubbleHelpCmd allows to create position dependent bubble help over the plot. For this, one sets bubbleHelpCmd to the value wt_PlotBubbleHelp[] and adds handling of the plot trigger _plot_bubblehelp. This trigger is generated by wt_PlotBubbleHelp with the mouse position information just before the bubble help pops up.

The following example code, shows how to handle the _plot_bubblehelp trigger with the function wt_PlotSetBubbleHelp and a custom function get_bubblehelp which creates the text and/or graphics object for the help to be shown. The bold sections need to be adapted to the specific plot widget content: 

  ...

    // Set attribute to create position dependent bubble help.

  WindowSetAttr [wkey, [plot: [bubbleHelpCmd: 'wt_PlotBubbleHelp[]']]];
  ...

    // Function 'get_bubblehelp' to generate bubble help.
    //  wkey:   window key
    //  widget: plot widget name
    //  x:      plot x value of the mouse position
    //  y:      plot y value of the mouse position
    //  loc:    1: mouse over x-axis, 2: mouse over y-axis, 3: mouse over data
    //  data:   additional custom data

  local function get_bubblehelp [wkey, widget, x, y, loc, data]
      local [dx,dy,r,rmin,rmax,w,h] = PlotPickInfo [wkey, widget, [x, y]];
      local text = twrite ['help text example - x: {} y: {}', x, y];
      local grkey = gr_create [];
      ...
      return [text, grkey];
  endfunction
  ...
  local [trigger, values] = WindowWait wkey;
  local data = [...];     // custom plot data, needed to create bubble help
  ...

    // Process _plot_bubblehelp trigger.

  if trigger == '_plot_bubblehelp' then
     wt_PlotSetBubbleHelp [wkey, trigger, values.(trigger), #get_bubblehelp, data];
  endif
  ...
