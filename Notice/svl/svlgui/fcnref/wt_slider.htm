Slider Widget

Description

Slider for entering numeric data. The precision of the numeric value is determined by incr in the range attribute. Slider is identical to Scale except that no text input field is present.

Attributes

Slider inherits all core attributes and defines the following attributes:

Attribute Description Values Default
width Size of the slider in character widths 1 to 255 10
vertical Orientation of slider 1 = vertical
0 = horizontal		 0
range Data increment and range multipliers: [minfac,maxfac,incr]

The actual data range is given by [minfac*incr,maxfac*incr]; the end values are included in the range.

In scrollbar mode, an optional fourth value thumbfac is accepted, which sets the width of the scrollbar thumb. The actual data range of the scrollbar is given by [minfac*incr,(maxfac-thumbfac)*incr].

sliderStyle Visual style and behavior of the slider widget.

In scrollbar mode, the slider looks and behaves like a scrollbar, and the width of the scroll thumb represents the size of the area to be scrolled. The thumb width is set as the fourth value of the range attribute.

 'slider', 'scrollbar' 'slider'
sliderValue Show the current slider value below the slider thumb. A mouse click on the value opens a popup in which one can change the slider value. 1 (True), 0 (False) 0
enableWheel Enable scrolling with the mouse wheel 1 (True)
0 (False)		 1

Trigger

Any mouse drag or button press in the slider which causes the sliders value to change.

Data Value

The number corresponding to the current slider position.
