Pager Widget

Description

Pager is a manager widget that displays only one of its children at a time, each on its own separate "page." Pages are indexed from 1 to the number of children, and are ordered according to child creation order. The current page may be set via the page attribute. The size of the pager is the size of its largest child, regardless of which child is displayed.

Attributes

Pager inherits all core attributes and defines the following attributes:

Attribute Description Values Default
page Index of the page to display One-based page index 1

Trigger

None.

Data Value

The values of its children.
