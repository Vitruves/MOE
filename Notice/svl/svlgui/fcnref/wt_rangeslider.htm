RangeSlider Widget

Description

Double Slider for entering numeric range data. Two slider thumbs represent a value range with a high and low value. Each slider thumb can be moved separately. By dragging the connector between the two slider thumbs, both are moved in sync. The precision of the numeric value is determined by incr in the range attribute. RangeSlider is similar to Slider except that it represents a value range instead of a single value.

Attributes

RangeSlider inherits all core attributes and defines the following attributes:

width Size of the range slider in character widths 1 to 255 10
vertical Orientation of slider 1 = vertical
0 = horizontal		 0
range Data increment and range multipliers: [minfac,maxfac,incr]

The actual data range is given by [minfac*incr,maxfac*incr]; the end values are included in the range
