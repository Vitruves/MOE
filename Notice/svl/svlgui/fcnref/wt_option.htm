Option Menu Widget

Description

A pull-down menu of options of which exactly one is selected at any given time. The name of the selected option appears in the widget.

Attributes

Option inherits all core attributes and defines the following attributes:

Attribute Description Values Default
text Pull-down menu item labels Vector of tokens []
columns Number of columns for layout 1 to 127 1
type Return value data type 'char', 'int' 'char'
minWidth Minimum width (in characters) of each of the menu items 0 to 127 0
foreground Option label text color.

Set a vector of colors to color each item in the pull-down menu separately.

Note! The Option label itself inherits the attribute from the currently selected menu item The vector length must match the number of menu items (text) and a foreground vector must be specified after the text attribute.

WINDOW_COLORS 'foreground'
background Option label background color.

Set a vector of colors to color each item in the pull-down menu.

See foreground attribute for details.

WINDOW_COLORS 'background'
font Option label text font.

Set a vector of fonts to have different text fonts for each item in the pull-down menu.

See foreground attribute for details.

WINDOW_FONTS 'medium'
sensitive Option sensitivity.

If a vector of sensitivities is specified, each item in the pull-down menu is enabled/disabled separately. Disabled menu items can not be selected.

See foreground attribute for details.

WINDOW_COLORS 'foreground'
graphics A vector of graphics object keys (see gr_create) or graphics content (see gr_data). The number of elements in the vector defines the number of buttons, and must be consistent with the text attribute. Vector of keys or content. 0
grPosition Layout of graphic object with respect to text, when both are present 'left', 'right', 'above', 'below', 'overlay' 'left'
grScale Scaling policy for the graphic object 'none', 'anisotropic', 'isotropic', 'isotropic-shrink' 'isotropic'
bubbleHelp Option label bubble help text.

Set a vector of bubble helps to have different help text for each item in the pull-down menu.

See foreground attribute for details.

token ''
bubbleHelpGraphics Option label bubble help graphics object.

Set a vector of bubble helps graphics to have different help graphics for each item in the pull-down menu.

See foreground attribute for details.

Vector of keys or content. 0

Trigger

A change in the option menu displayed item. Selecting the same item will not cause a trigger.

Data Value

The value of the option menu is returned as a token or one-based index (depending on the state of the type attribute) denoting the currently selected menu item. The default value is the first item.
