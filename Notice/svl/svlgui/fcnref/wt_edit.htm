Edit Widget

Description

The Edit widget is a scrollable multi-line text editor.

Attributes

Edit inherits all core attributes and defines the following attributes:

Attribute Description Values Default
len Number of lines displayed 1 to 255 25
width Number of characters displayed per line 1 to 255 80
type Edit return value data type 'char','modint' 'char'
cursorPos Text cursor position and text selection as vector of two integer numbers. The first number is the cursor position and the second the length of the selection starting at the cursor position; negative for a selection to the left, or positive to the right. [number, number] [0,0]

Trigger

None, if the data type is char.
If the data type is modint, the widget will trigger when the editor text, text selection or cursor position changes.

Data Value

If the data type is char the editor's text is returned as a token.
If the data type is modint a vector with the editor's text, the cursor position and selection is returned: 

   text                            // type = 'char'
   [text, cursor_pos, sel_len]     // type = 'modint'
