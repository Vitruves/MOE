Radio Box Widget

Description

Radio box, a set of toggle buttons, of which exactly one is selected at any given time.

Attributes

Radio inherits all core attributes and defines the following attributes:

Attribute Description Values Default
text String labels Vector of tokens []
columns Number of columns for layout 1 to 127 127
type Return value data type 'char', 'int' 'char'
minWidth Minimum width (in characters) of each of the buttons 0 to 127 0
sensitive Enable/disable radio button(s). If a vector of sensitivities is specified, each radio button is enabled/disabled individually. The vector length must match the number of buttons (text). 0, 1 1
foreground Button label text color. If a vector of colors is specified, each radio button is colored individually. The vector length must match the number of buttons (text). WINDOW_COLORS 'foreground'
background Button label background color. If a vector of colors is specified, each radio button is colored individually. The vector length must match the number of buttons (text). WINDOW_COLORS 'background'
font Button label text font. If a vector of fonts is specified, each radio button uses an individual font. The vector length must match the number of buttons (text). WINDOW_FONTS 'medium'
alignment Horizontal alignment of text 'left', 'center', 'right' 'center'
graphics A graphics object key (see gr_create) or graphics content (see gr_data) Scalar key, or vector content 0
grPosition Layout of graphic object with respect to text, when both are present 'left', 'right', 'above',
'below', 'overlay'
'flushLeft', 'flushRight',
'flushAbove', 'flushBelow' 		'left'
grScale Scaling policy for the graphic object 'none', 'anisotropic', 'isotropic', 'isotropic-shrink' 'none'

Trigger

A change in the selected button within the radio box.

Data Value

The last button pushed. If character return data is specified, the value is the name of the radio button (a token), otherwise it is the index (starting from 1) of the selected item. If no button has ever been pushed, it defaults to the first button specified in the window descriptor.
