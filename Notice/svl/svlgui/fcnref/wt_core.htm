Attributes

All widgets support the following attributes. In general, 0 means the attribute is on, 1 off.

Attribute Description Values Default
background Background color WINDOW_COLORS 'none' (Widget dependent)
bubbleHelp Text to display when cursor is motionless on widget Vector of tokens []
bubbleHelpGraphics Add graphic object to the bubble help Scalar key, or vector content: Graphic object 0
centerH Center widget horizontally within its cell 0, 1 0
centerV Center widget vertically within cell; vertical (X-coordinate) alignment point ignored 0, 1 0
collapse Collapse widget visually. A collapsed widget is not visible, and does not occupy any space in the layout. 0, 1 0
extendH Widget always stretched horizontally to fill width of cell 0, 1 0
extendV Widget always stretched vertically to fill height of cell 0, 1 0
flushLeft Widget placed at left edge of cell; vertical (Y-coordinate) alignment point ignored 0, 1 0
flushTop Widget placed at top edge of cell; horizontal (X-coordinate) alignment point ignored 0, 1 0
font Widget font WINDOW_FONTS 'medium'
foreground Foreground color WINDOW_COLORS 'none' (Widget dependent)
margin Margin (extra space) added around widget 0, 1 0
name Name to be used for referencing the widget; must be unique within a window descriptor to allow widget to be unambiguously referenced token []
onTrigger Change to widget's value triggers window WINDOW_ON_TRIGGER 'ignore'
sensitive Widget and all its children are disabled 0, 1 1
shadow Widget displayed with shadow WINDOW_SHADOWS 'none'
title Widget title token []
titleFont Widget title font WINDOW_FONTS 'mediumBold'
titleTop Title placed on top rather than at left 0, 1 0

Notes:

  • Unit extension applies when setting the onTrigger attribute of widgets that are composed of several components that can each trigger, e.g. a set of pushbuttons.
  • The foreground and background colors may not be applicable to some widgets. In such cases, they will be ignored.
