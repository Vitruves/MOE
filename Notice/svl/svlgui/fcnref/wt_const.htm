Window Constants

Description

The SVL Window Toolkit defines the following constants:

WINDOW_OBJECTS Contains the complete list of widgets specified as tokens.
WINDOW_COLORS Tokens that specify the color names used by the foreground and background attributes. Color names are as follows:

Widget Type Colors (adapt dynamically to MOE Appearance):
'foreground', 'background', 'textForeground', 'textBackground', 'highlight', 'highlightForeground', 'canvasForeground', 'canvasBackground'

Default Widget Color (adapts dynamically to MOE Appearance and widget type):
'none'

Base Foreground Colors For Colored Text (adapt dynamically to MOE Appearance):
'baseRed', 'baseGreen', 'baseBlue', 'baseYellow', 'baseCyan', 'baseMagenta', 'basePurple', 'baseOrange', 'baseBrown', 'basePink'

Primary Colors:
'black', 'white', 'red', 'green', 'blue', 'yellow', 'cyan', 'magenta', 'purple', 'orange', 'brown', 'pink', 'gray', 'lightGray', 'darkGray', 'darkRed', 'darkGreen', 'darkBlue', 'darkYellow', 'darkCyan', 'darkMagenta', 'darkPurple', 'darkOrange', 'darkBrown', 'darkPink'
WINDOW_DATA_TYPES Tokens specifying the available data type attribute:
  • 'int': integers only
  • 'char': token
  • 'real': any numeric
WINDOW_FONTS Tokens specifying the font names used by the font attributes. Font names are as follows:

Proportional Fonts: 'small', 'smallBold', 'medium', 'mediumBold', 'large', 'largeBold'

Fixed Fonts: 'smallFixed', 'smallFixedBold', 'mediumFixed', 'mediumFixedBold', 'largeFixed', 'largeFixedBold'
WINDOW_FSB_MODES Tokens specifying the FSB and FSBText validation modes:
  • 'none': no validation
  • 'open': ensure that filename exists
  • 'saveAs': ensure that user wishes to overwrite existing file
WINDOW_ON_TRIGGER Tokens specifying the onTrigger attribute settings:
  • 'return': perform no validation
  • 'validate': validate all widgets when triggered
  • 'ignore': do not return from WindowWait when triggered
  • 'exit': kill task when triggered
WINDOW_SHADOWS Tokens specifying the shadow attribute settings:
  • 'none': no shadow
  • 'noline': space allocated, but no line is drawn
  • 'in': inset look
  • 'out': raised look
  • 'etched-in': inset etched look
  • 'etched-out': raised etched look
  • 'outline': solid outline
  • 'rounded-in': rounded inset look
  • 'rounded-out': rounded raised look
WINDOW_SPACING Tokens specifying the spacing attribute: 'none', 'small', 'medium', 'large'
