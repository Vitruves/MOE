Checkbox Widget

Description

A single toggle button that maintains a boolean state (True/False).

Attributes

Checkbox inherits all core attributes and defines the following attributes:

Attribute Description Values Default
text Checkbox label token []
foreground Label text color WINDOW_COLORS 'foreground'
background Label background color WINDOW_COLORS 'background'
font Label text font WINDOW_FONTS 'medium'
alignment Horizontal alignment of text 'left', 'center', 'right' 'left'
graphics A graphics object key (see gr_create) or graphics content (see gr_data). Scalar key, or vector content 0
grPosition Layout of graphic object with respect to text, when both are present 'left', 'right', 'above',
'below', 'overlay'
'flushLeft', 'flushRight',
'flushAbove', 'flushBelow' 		'right'
grScale Scaling policy for the graphic object 'none', 'anisotropic', 'isotropic', 'isotropic-shrink' 'isotropic'
checkboxStyle Change the visual style of the checkbox. See $MOE/sample/windemo.svl for examples. A token matching any of these:
'checkMark', 'radioButton', 'roundRadioMark', 'squareMark', 'crossMark', 'checkMarkRadioButton', 'checkMarkButton'.		 'checkMark'

Trigger

A change in state of the checkbox.

Data Value

1 if set, 0 if unset.
